# Bot_Inviter Sigma26
Sigma26 Bot
bot do wysyłania zaproszeń na grupe Sigma26

-------------------------------------------
Jak korzystać z Bot_Invater
-------------------------------------------
1. zaloguj sie (Nick/e-mail i hasło)
2. w polu nazwa użytkownika wpisz nazwe użytkownika którego zapraszasz
3. naciśnij przycisk send


Jeżeli wszystko wpisałeś poprawnie zaproszenie zostało wysłane i na ekranie nie pojawił sie żaden błąd
